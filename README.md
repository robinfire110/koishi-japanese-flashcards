# Koshi - Japanese Kana Flashcards (koishi-japanese-flashcards)
Japanese Hiragana & Katakana Flashcards for all Pebble watches (OG, Time, Round, Duo and Time 2)!

# Rebble App Store Link
https://apps.rebble.io/en_US/application/64177b1e419fd9053e31a7b5

# Features
This watchapp gives you a full set of Hiragana and Katakana flashcards at your wrist. Study your kana recognition anytime, anywhere!

- Fully customizable deck options (size and kana)
- Pre-made deck presets
- 208 Hiragana & Katakana characters
- Fully image-based, no additional language pack needed

If you wish to install this watchface, please do so through the Rebble App Store link above. The Pebble documentation is all over the place since they shut down, so feel free to use code from this repo to help you in your own projects. You're free to reference and use this code, but please don't take the whole project and upload it as your own.
